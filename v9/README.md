# v9 算法详解（深度优化续航版）

## 1. 版本定位

`v9` 是在 `v8` 完成 5 轮受控迭代后的继续深度优化版本。目标是在不重新大规模调参的前提下，进一步提升主指标 `R²` 与次指标 `RMSE`。

核心策略：

- 复用 `v8` 已验证的三类候选模型（`XGBoost_v8` / `LightGBM_v8` / `HGB_v8`）；
- 引入 `HGB_v7_baseline` 作为稳定锚点；
- 从全局单权重融合升级为**龄期分段自适应融合**（`age <= 28` 天与 `age > 28` 天使用不同权重）。

---

## 2. 方法总览

### 2.1 候选模型池

- `XGBoost_v8`（v8 Iter-2 最优参数）
- `LightGBM_v8`（v8 Iter-3 最优参数）
- `HGB_v8`（v8 Iter-1 最优参数）
- `HGB_v7_baseline`（v7 强基线参数）

### 2.2 双特征空间融合

- `v8` 模型使用扩展机理特征空间；
- `HGB_v7_baseline` 使用 v7 特征空间；
- 通过 OOF 预测在同一目标空间上做融合。

### 2.3 融合策略

1. **Global Blend**：全样本共享一组权重；
2. **Age-aware Piecewise Blend**：
   - 早龄期：`age <= 28`
   - 后龄期：`age > 28`
   - 两段分别优化权重。

---

## 3. 结果对比

### 3.1 v9 内部策略对比

| 策略 | R²_mean | RMSE_mean |
|---|---:|---:|
| Global Blend | 0.948724848 | 3.700053455 |
| **Age-aware Piecewise** | **0.948755187** | **3.699571200** |

最终选择：`age_piecewise`。

### 3.2 对比 v8 / v7

| 指标 | v7 | v8 | v9 | v9-v8 变化 | v9-v7 变化 |
|---|---:|---:|---:|---:|---:|
| R²_mean | 0.947965399 | 0.948724848 | **0.948755187** | +0.000030339 | +0.000789788 |
| RMSE_mean | 3.740781572 | 3.700053455 | **3.699571200** | -0.000482255 | -0.041210372 |

结论：`v9` 在 `v8` 基础上继续小幅稳定提升，且相对 `v7` 维持双指标显著正增益。

---

## 4. 关键权重解释

### 4.1 全局权重

- `XGBoost_v8`: 0.343245
- `LightGBM_v8`: 0.025428
- `HGB_v8`: 0.041109
- `HGB_v7_baseline`: 0.590218

### 4.2 分段权重（v9 最终）

**早龄期（<=28 天）**
- `XGBoost_v8`: 0.313813
- `LightGBM_v8`: 0.039929
- `HGB_v8`: 0.013239
- `HGB_v7_baseline`: 0.633019

**后龄期（>28 天）**
- `XGBoost_v8`: 0.463224
- `LightGBM_v8`: ≈0
- `HGB_v8`: 0.122167
- `HGB_v7_baseline`: 0.414610

工程解释：早龄期更依赖稳健锚点，后龄期增加 XGBoost 权重以刻画复杂增长非线性。
